# Mind. — PWA

Maquette interactive installable (Progressive Web App) de l'application Mind.

## Mise en ligne (GitHub Pages)
1. Créer un dépôt public, ex. `mind-app`.
2. Téléverser TOUS les fichiers de ce dossier à la racine du dépôt.
3. Settings → Pages → Source : `Deploy from a branch` → Branch : `main` / `/root` → Save.
4. Attendre 1–2 min. L'URL publique apparaît en haut de la page Pages.

## Installation sur téléphone
- iPhone (Safari) : ouvrir l'URL → bouton Partager → « Sur l'écran d'accueil ».
- Android (Chrome) : ouvrir l'URL → menu ⋮ → « Installer l'application » / « Ajouter à l'écran d'accueil ».

Fichiers : index.html, manifest.json, sw.js, icon-192.png, icon-512.png, icon-maskable-512.png, apple-touch-icon.png
